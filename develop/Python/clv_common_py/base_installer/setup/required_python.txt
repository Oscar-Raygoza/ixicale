==========================
Unsupported Python version
==========================
This version of Flask requires Python {}.{}, but you're trying to
install it on Python {}.{}.
This may be because you are using a version of pip3 that doesn't
understand the python_requires classifier. Make sure you
have pip3 >= 9.0, then try again:
$ python3 -m pip3 install --upgrade pip3

This will install the latest version of Flask which works on your
version of Python. If you can't upgrade your pip3 (or Python), request
an older version of Flask:
$ python3 -m pip3 install Flask
